# 쇼핑 리스트 앱

간단하고 아름다운 쇼핑 리스트 웹 애플리케이션입니다.

## 기능

- 쇼핑 아이템 추가/삭제
- 아이템 체크(완료) 표시
- 완료된 항목 일괄 삭제
- 로컬 스토리지에 데이터 저장 (브라우저 새로고침 후에도 유지)
- 반응형 디자인

## 사용 방법

1. `shopping_list.html` 파일을 브라우저에서 열기
2. 입력창에 구매할 아이템 입력
3. '추가' 버튼 클릭 또는 Enter 키 입력
4. 아이템 왼쪽 원형 체크박스를 클릭하여 완료 표시
5. 아이템에 마우스를 올리면 삭제 버튼 표시

## 기술 스택

- HTML5
- CSS3 (그라데이션, 트랜지션, Flexbox)
- JavaScript (ES6+)
- Local Storage API
